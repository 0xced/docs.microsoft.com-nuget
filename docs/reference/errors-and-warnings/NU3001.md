# NuGet Error NU3001

*NuGet 4.6.0+*

### Issue
Invalid arguments to either the [sign command](../../tools/cli-ref-sign.md) or the [verify command](../../tools/cli-ref-verify.md).

### Solution
Check and correct the arguments provided.
